The Roadmap

Full-Stack Curriculum

A step-by-step breakdown of your 24-week journey from absolute beginner to hired professional.

Phase 1 Weeks 1 - 4

Web Fundamentals & UI

We start from scratch. You will learn how the internet works and build your first responsive websites from the ground up.

  • HTML5 Semantic Tags & Accessibility
  • CSS3 Flexbox, Grid, & Animations
  • Responsive Web Design (Mobile-First)
  • Git, GitHub, and Terminal Basics
Phase 2 Weeks 5 - 10

JavaScript Mastery

This is where things get interactive. You will learn the core programming language of the web and start writing actual logic.

  • Variables, Loops, Functions & Objects
  • DOM Manipulation & Event Listeners
  • ES6+ Modern Syntax
  • Asynchronous JS (Promises, Async/Await) & Fetching APIs
Phase 3 Weeks 11 - 16

Front-End Frameworks (React)

Transition from vanilla JavaScript to React, the industry standard for building massive, dynamic user interfaces.

  • React Fundamentals & JSX
  • Components, Props, and State
  • React Hooks (useState, useEffect)
  • Routing & Global State Management (Redux/Context API)
Phase 4 Weeks 17 - 21

Back-End & Databases

Learn to write server-side code, manage databases, and build your own secure RESTful APIs.

  • Node.js & Express.js Servers
  • RESTful API Architecture
  • MongoDB & Mongoose (NoSQL Databases)
  • User Authentication (JWT, bcrypt)
Phase 5 Weeks 22 - 24

Final Project & Career Prep

Build a massive, production-ready Full-Stack application from scratch and prepare for technical interviews.

  • Capstone Project Development
  • Deployment (Vercel, Render, AWS)
  • Resume & Portfolio Polish
  • Mock Technical Interviews & Algorithm Practice
